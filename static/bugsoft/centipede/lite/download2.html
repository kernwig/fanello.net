How to Download

Browsers don't know what to do with the .lbr files. In most cases, if you just click on the file your browser will attempt to display it to you. What you want to do is download the files.

How this is done depends on your browser and operating system:

  • Windows 95/98/NT - right click on the file and choose "Save Link As..."

  • Xwindows - right click on the file and choose "Save Link As..."

  • Macintosh - block the file and hold the mouse button down until a menu comes up. Choose "Save Link As..."

  • Lynx - Highlight the file and press D to download.
