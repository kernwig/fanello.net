Centipede is a revolutionary new BBS software package for the Commodore 128 computer system, taking advantage of the latest in computer hardware and software technologies. It represents over eight years of BBS programming experience on behalf of its author, covering four different systems.
His second BBS program, author Adam Fanello and Bugsoft are proud to unleash Centipede BBS!

  • Features
    • Runs on a Commodore 128 in C128, 80 column mode (2 MHz)
    • Supports LtK, CMD, and all other C= Drives
    • Supports from 300 to 230,400 bps
    • Graphics - 80 column C/G, 40 column C/G, ANSI, ASCII, Superes, and RipScript
    • Supports up to 15 separate Message Boards
    • Each Message Board supports 26 threaded or unthreaded categories.
    • Supports up to 15 U/D Categories with 26 directories each, for up to 254 directories, with 300 files per directory. (That's over 76 thousand files!)
    • Simulated 40 column display when C64 users are online
    • Scripting
    • Polymorphic Menu System
    • Unlimited online Gaming abilities. (More added all the time!)
    • Protocols Xmodem, Xmodem 1k, Punter, Multi-Punter, Ymodem-batch.
    • Easy to install add-on features
    • ComLink and CommNet - International Messaging Network
    • Net64 (Color 64 Network)
    • Online Help Files
    • Extensive message board editing and control.
    • Message Quoting
    • E-mail file attachments
    • Superior SysOp control features that are built in and EASY to use.
    • Local Buffer
    • Built in Term program
    • Multiplexing on a LtK dual line system.
    • Cross-line chat capable with Multiplexed systems
    • Individualized user access control.
    • Standard, Word-Wrapped, Colored, and Split Screen chat modes
    • Automatic detection of peripherals (CMD HD, RamLink, 1541, LtK, etc.)
    • Full Screen Text Editor
    • Detailed Caller Log
    • Very fast system all the way around.
    • Full Screen U/D File Selector
    • Very programmer friendly; easy to write your own add-ons

    • Nine access levels, with individual adjustments available.
    • Automatic accounts purging
    • Customizable new member application
    • Up to 1000 lines per message
    • Local mode use password
    • Error recovery and reporting module
